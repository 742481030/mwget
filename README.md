## mwget 

一个多进程下载工具，可并发下载以加快速度

### 介绍

安装 【 pypi更换域名，导致新版本没法提交上去. 】
```
pip install mwget
```

使用mwget下载
```
python -m mwget.multiwget http://mirrors.163.com/centos/7/isos/x86_64/CentOS-7-x86_64-NetInstall-1611.iso
```

or
```
git clone git@github.com:rfyiamcool/mwget.git
cd mwget
python test.py url
```
